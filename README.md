# rcpt\_to-strip-tagging

[![Build Status][ci-img]][ci-url]
[![NPM][npm-img]][npm-url]

Strip tagging from inbound rcpt\_to addresses

## Installation

```sh
cd /etc/haraka
npm i haraka-plugin-rcpt_to-strip-tagging
echo 'rcpt_to-strip-tagging' >> config/plugins
```
